# Aprende UltraTorrent

Todo aquí se construye sobre lo anterior. Si eres nuevo, léelo en orden — el
camino completo toma como una tarde, y la mayor parte es esperar por descargas.

## El camino

| Paso | Página | Lo que obtienes |
| --- | --- | --- |
| 1 | [**Inicio Rápido**](/learn/quick-start) | Una instalación funcionando y una descarga completada, en ~15 minutos. |
| 2 | [Conceptos Básicos](/learn/concepts) | El vocabulario que el resto de la documentación asume: motores, indexadores, listas de seguimiento, perfiles. |
| 3 | [Resumen de Arquitectura](/learn/architecture-overview) | Qué hace cada contenedor, y cuáles puertos importan. |
| 4 | [Mi Primera Descarga](/learn/first-download) | El ciclo de vida completo, una pantalla a la vez — agregar, descargar, identificar, organizar, notificar. |
| 5 | [Flujos de trabajo](/learn/workflows) | Cómo encajan las piezas de punta a punta, en diagramas. |
| 6 | [Tutoriales](/learn/tutorials/) | De principiante a empresarial, cada uno una configuración completa que funciona. |

:::tip Lee Conceptos Básicos aunque tengas prisa
La mayoría de las preguntas de soporte son en realidad preguntas de vocabulario. Diez minutos en
[Conceptos Básicos](/learn/concepts) te ahorrarán una hora en
[Solución de problemas](/operate/troubleshooting).
:::

- **Llámalo desde tu propio código** → [Referencia de la API REST](/reference/api).
- **Algo se rompió** → [Solución de problemas](/operate/troubleshooting), organizado por síntoma.
